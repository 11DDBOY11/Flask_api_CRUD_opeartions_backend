# Flask_api_CRUD_opeartions_backend
This project is a RESTful Student Management API built using Flask and SQLite. The application demonstrates basic CRUD (Create, Read, Update, Delete) operations for managing student records in a database.

The API allows users to add new students, retrieve all student records, fetch a specific student by ID, update student details, and delete student records. It also includes email validation to prevent duplicate entries in the database.

The backend is implemented using Flask, with SQLite as the database. The project follows a modular structure where database connection logic is separated into a dedicated file for better maintainability and scalability.

This project helped me understand REST API design, HTTP methods (GET, POST, PUT, DELETE), database integration, and JSON response handling in Flask. It is suitable for beginners who want to learn backend development using Python and Flask.

Future improvements may include authentication, input validation, error handling, and deployment to cloud platforms .
